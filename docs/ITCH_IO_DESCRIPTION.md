# itch.io Description Draft

## Short Description (64 words)

Flappy Something begins as a shamelessly cheap bird game: bright ads, impossible promises, and a Gate 40 nobody can cross. Following a disputed high score leads you through an obsolete phone, erased company records, and a lost version called Skyline 256. It is a story about preservation, the people behind a game, and whether a game can remain playable after support ends for everyone.

## Basic Controls (61 words)

Click or tap inside Flappy Something to flap. Press Space for the same action; Space also starts a new run after a crash. Use the phone apps with mouse or touch: search, open messages, inspect records, and follow the evidence at your own pace. Later, collect every glowing route point in one flight before Gate 40. The game saves progress locally.
